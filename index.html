Configuração GitHub

O token é necessário apenas para fazer upload de novas planilhas. Qualquer pessoa já consegue visualizar os dados sem precisar configurar nada. O token fica salvo somente neste navegador.
Como gerar o token (só quem vai fazer upload precisa):

1
Acesse github.com/settings/tokens/new
2
Em Note escreva algo como aguia-site
3
Em Expiration escolha No expiration
4
Marque a permissão repo (primeira opção)
5
Clique em Generate token e cole o valor abaixo

Para habilitar o upload de planilhas, informe seu token. Clique em ⚙ configurações e insira o Personal Access Token do GitHub. Visitantes comuns não precisam configurar nada — os dados carregam automaticamente.
